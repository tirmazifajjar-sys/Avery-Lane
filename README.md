# Avery Lane — UI/UX Designer Portfolio

A modern, responsive portfolio website for a fictional UI/UX designer. Built with HTML, CSS, and JavaScript, the site showcases premium UI design through editorial layouts, smooth animations, immersive imagery, and thoughtful typography.

Designed as a single-page experience, the portfolio highlights featured projects, design process, testimonials, and contact information while maintaining excellent responsiveness and performance.

---

## ✨ Features

- Modern editorial-inspired interface
- Fully responsive across desktop, tablet, and mobile
- Full-screen hero section
- Animated services ticker
- Interactive portfolio case studies
- Embedded studio reel
- Design process timeline
- Client testimonial section
- Call-to-action contact section
- Scroll reveal animations
- Smooth scrolling navigation
- Accessibility support with reduced motion preferences

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (DM Sans & Fraunces)

## ⚡ Animations

The website uses the **Intersection Observer API** to trigger lightweight reveal animations as users scroll.

Animations include:

- Fade-in effects
- Vertical motion transitions
- One-time reveal animations for improved performance

---

## 📱 Responsive Design

Optimized for:

- Desktop
- Laptop
- Tablet
- Mobile

Responsive layouts are achieved using CSS media queries that adapt typography, navigation, grids, spacing, and media elements across screen sizes.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/avery-lane-portfolio.git
```

Navigate into the project folder:

```bash
cd avery-lane-portfolio
```

Open `index.html` in your preferred browser, or start a local server:

```bash
python -m http.server
```

or

```bash
npx serve
```
